Adding a project

Create a new project by clicking on "New Project" in the nav bar

Adding a task

Double click on a project to bring up the "New Task" popup. Fill in the Name and click "New Task" to add the task to the project

Editing a Project

Double click on the Project Name to edit its name

Editing a Task

Double click on the task name, description or priority to edit the chosen property. Dates can be changed by clicking on the date selection box

Finishing a Task

Click on the finished task to cross it off

Deleting a Task

Right click on the chosen task to delete it.

Deleting a Project

Right click on the project (must not hover over a task) to delete it
